U-LEAF UI Standards & Guidelines 2.0
U-LEAF User Interface Standards & Guidelines will facilitate teams to maintain, update and expand the application while retaining stylistic, organizational, and technical uniformity throughout. This document should be used as a reference tool and complements U-LEAF Templates.
